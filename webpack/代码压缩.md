# 代码压缩
## 安装
```
    npm i -D uglifyjs-webpack-plugin
``` 
## 配置
```js
    const UglifyJSPlugin = require('uglifyjs-webpack-plugin');
    
    module.exports = {
        plugins: [
            new UglifyJSPlugin(),
        ]
    }
```
### 低版本
```js
    var webpack = require('webpack');

    module.exports = {
        plugins: [
            new webpack.optimize.UglifyJsPlugin(),
        ],
    }
```
